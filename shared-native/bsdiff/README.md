# Original bsdiff/bspatch Implementation

This folder contains the original bsdiff and bspatch implementations by Colin Percival.

## Files

- `bsdiff.c` - Original bsdiff implementation (404 lines)
- `bspatch.c` - Original bspatch implementation (204 lines)

## Source

- **Author:** Colin Percival
- **Source:** Debian package repository (version 4.3-15)
- **Format:** BSDIFF40 (standard, widely compatible)

## Format

Uses standard BSDIFF40 format with bzip2 compression. Compatible with all standard bsdiff tools.

## Dependencies

- Requires bzip2 library (`libbz2`)
- Standard C library

## Usage

These files will be used in test scripts and can be integrated into iOS, Android, and CLI applications.
